# Turtle Crossing — A Python Arcade Game

**Turtle Crossing** is a simple arcade-style game built with Python's `turtle` graphics module. Inspired by classic Frogger mechanics, the player controls a turtle trying to cross a busy road filled with speeding cars. Each successful crossing increases the difficulty, making it a fun and progressively challenging experience.

## Gameplay Features

- Player-controlled turtle using arrow key "UP"⬆️  
- Randomized car generation and movement  
- Collision detection and game-over logic  
- Level progression with increasing speed  
- Modular design using `Player`, `CarManager`, and `Scoreboard` classes
